# overpeek-engine
Engine project

### Cloning
```
git clone --recursive https://github.com/Overpeek/overpeek-engine.git
```


### CMake
```
cmake -B build/
```
#### Depends:
- Freetype
- GLFW3
- OpenGL
- OpenAL
